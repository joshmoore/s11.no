53-rda-f1-01d-t1-data-no-pid

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=item to https://s11.no/2022/a2a-fair-metrics/53-rda-f1-01d-t1-data-no-pid/files/1 (text/csv, type attribute declared, w3id PID)

This is a test for FAIR principle RDA-1-01D - Data is identified by a persistent identifier.

Note:This is a more usual setup than 53-rda-f1-01d-t1-data-no-pid, where the data item is a regular URI, as s11.no is not a known PID provider. However, to be "PID-like", this example uses a HTTP 302 Found redirect to the final file test-apple-data.csv.
