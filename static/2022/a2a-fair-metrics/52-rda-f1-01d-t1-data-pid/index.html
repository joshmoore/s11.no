52-rda-f1-01d-t1-data-pid

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=item to https://w3id.org/a2a-fair-metrics/52-rda-f1-01d-t1-data-pid/files/1 (text/csv, type attribute declared, w3id PID)

This is a test for FAIR principle RDA-1-01D - Data is identified by a persistent identifier.

Note:This is a bit unusual setup where the data file itself has a separate PID, which do not include file extension.

The actual URI is not exposed by the Signposting headers. In this example the full file name with extension 1.csv is hidden by the web server URL, but will be exposed by the Content-Location header, which in many browsers are reflected in the download filename.
