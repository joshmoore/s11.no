23-http-citeas-describedby-item-license-type-author

This landing page has all required and optional FAIR signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=describedby to index.ttl (with type="text/turtle" attribute declared)
  • rel=item to CSV file
  • rel=license to SPDX identifier
  • rel=type to Dataset
  • rel=author to ORCID

For completeness, there's also a collection link from the CSV file back to this HTML landing page, but its license etc. are declare here.
