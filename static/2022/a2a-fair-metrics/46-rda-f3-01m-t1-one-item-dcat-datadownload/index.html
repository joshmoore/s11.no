46-rda-f3-01m-t1-one-item-dcat-datadownload

This landing page has three signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=describedby to dataset.ttl (text/turtle, type attribute declared)
  • rel=item to CSV file (text/csv, type attribute declared)

This is a test for FAIR principle RDA-F3-01M - Metadata includes the identifier for the data.

Test 1: each item appears in one or more metadata files (describedby) as an identifier property, and string-match of item against identifier.

The Turtle file has 6 statements using the namespace http://www.w3.org/ns/dcat#, including a locally identified node for the Distribution which has the data identifier as an IRI reference from downloadURL. Note that the DCAT Dataset in this example deliberately do not include the PID.
