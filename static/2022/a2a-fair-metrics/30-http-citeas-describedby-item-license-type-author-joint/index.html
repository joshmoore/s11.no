30-http-citeas-describedby-item-license-type-author-joint

This landing page has all required and optional FAIR signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=describedby to index.ttl (with type="text/turtle" attribute declared)
  • rel=item to CSV file
  • rel=license to SPDX identifier
  • rel=type to Dataset
  • rel=author to ORCID

This test is identical to 23-http-citeas-describedby-item-license-type-author except that all these Link relations are in a single HTTP header joint with , (the non-signposting link rel=stylesheet may be listed separately). Additionally this particular test case does not use "quotes" on the link relations.
