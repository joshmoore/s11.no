47-rda-f3-01m-t1-multiple-item-dcat-datadownload

This landing page has three signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=describedby to dataset.ttl (text/turtle, type attribute declared)
  • rel=item to CSV file (text/csv, type attribute declared)
  • rel=item to OpenOffice Spreadsheet (application/vnd.oasis.opendocument.spreadsheet, type attribute declared)

This is a test for FAIR principle RDA-F3-01M - Metadata includes the identifier for the data.

Test 1: each item appears in one or more metadata files (describedby) as an identifier property, and string-match of item against identifier.

The Turtle file has 9 statements using the namespace http://www.w3.org/ns/dcat#, including two locally identified nodes for the Distribution which has the data identifier as an IRI reference from downloadURL for each item above. Note that the DCAT Dataset in this example deliberately do not include the PID.
