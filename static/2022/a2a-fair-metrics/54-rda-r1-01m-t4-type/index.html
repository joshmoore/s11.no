54-rda-r1-01m-t4-type

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=type to https://schema.org/CreativeWork

This is a test for FAIR principle RDA-R1-01M - Plurality of accurate and relevant attributes are provided to allow reuse.

Test 5: Each rel=type points to a URL.

In this case there is no separate downloadable item, the content (this test) is contained in the page (like an open access journal article), thus this is not typed as an AboutPage.
