68-rda-r1-01m-t1-some-item-type

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=item to CSV file (type=text/csv)
  • rel=item to OpenOffice Spreadsheet (no type)

This tests RDA-R1-01M, test 1: every rel=item link has a type="" attribute by having multiple files. This is a negative test, because test-apple-data.ods signposting should also have a type.
