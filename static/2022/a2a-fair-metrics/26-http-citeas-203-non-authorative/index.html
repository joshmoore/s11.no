26-http-citeas-203-non-authorative

This landing page has only one signposting HTTP header:

  • cite-as to w3id

The server will return status code 203 Non-Authoritative Information, e.g. as delivered by a proxy that may have transformed the content. The client may still consume the Link headers, but should present a warning to the user as the proxy may have rewritten the headers, as shown with the PID in this example.
