07-http-describedby-citeas-linkset-json

This landing page has three HTTP headers:

  • describedby to index.ttl (text/turtle, type attribute declared)
  • cite-as to w3id
  • linkset to linkset.json that again links to test-apple-data.csv and the above

Bonus: The linkset includes the optional rel=http://www.w3.org/ns/json-ld#context link so it is interpretable as JSON-LD.
