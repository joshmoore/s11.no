72-rda-r1-01m-t4-type-unregistered

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=item to Taverna workflow (type=application/vnd.taverna.t2flow+xml)

This tests RDA-R1-01M, test 4: The type/subtype is register with IANA even if it starts with vnd. or priv This is a positive for test 3, but negative for stricter test 4, as application/vnd.taverna.t2flow+xml was never registered.
