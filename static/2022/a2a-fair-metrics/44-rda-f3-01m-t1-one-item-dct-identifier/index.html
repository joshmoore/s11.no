This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=describedby to index.ttl (text/turtle, type attribute declared)

This is a test for FAIR principle RDA-F3-01M - Metadata includes the identifier for the data.

Test 1: each item appears in one or more metadata files (describedby) as an identifier property, and string-match of item against identifier.

In this case, the data identifier string https://s11.no/2022/a2a-fair-metrics/44-rda-f3-01m-t1-one-item-dct-identifier/test-apple-data.csv is referenced as a dct:identifier from the corresponding subject
