51-rda-f1-01m-t1-metadata-no-pid

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to URL
  • rel=describedby to https://s11.no/2022/a2a-fair-metrics/51-rda-f1-01m-t1-metadata-pid/metadata.ttl (text/turtle, type attribute declared)

This is a test for FAIR principle RDA-1-01M - Metadata is identified by a persistent identifier.

Note:This is a more typical setup where the metadata file itself has just a regular https URI, as s11.no is not a known PID provider. In this example the metadata file deliberately do not reference either PID.
