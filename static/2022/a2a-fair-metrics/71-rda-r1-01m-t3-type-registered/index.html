71-rda-r1-01m-t3-type-registered

This landing page has these FAIR Signposting HTTP headers:

  • rel=cite-as to w3id PID
  • rel=item to CSV file (type=text/csv)
  • rel=item to OpenOffice Spreadsheet (type=application/vnd.oasis.opendocument.spreadsheet)

This tests RDA-R1-01M, test 3: The type/subtype are registered in the IANA media type register This is a positive test because text/csv and application/vnd.oasis.opendocument.spreadsheet are both registered.
